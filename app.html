📅 Cronograma dos 3 empreendimentos
Prazo mínimo de 2 anos entre cada empreendimento. Todos concluídos até 31/12/2030.
➕ Adicionar / Atualizar empreendimento ▼
O que significa "legalizado"?
✅ Habite-se emitido pela prefeitura
✅ Registro no Cartório de Imóveis
✅ CNPJ ou CPF como proprietário formal
✅ IPTU regular e matrícula atualizada
💡 Empreendimento legalizado = patrimônio "bancável" — pode ser dado em garantia e vendido formalmente
🔄 Estratégia: transição gradual dos kits informais para ativos formalizados — sem sacrificar o caixa atual
⚠️ Risco atual: imóveis sem documentação não podem ser dados em garantia ou vendidos formalmente.
💡 Oportunidade: cada bem regularizado abre acesso a crédito com juros menores.
🎯 Objetivo: até 2030, ter os 3 empreendimentos 100% legalizados.

❓ Como funciona
📲 Compartilhar com a Jack (ou outro celular)
Gera um arquivo .html com todos os dados já dentro. A Jack abre e vê tudo igual ao seu celular — imóveis, gastos, aluguéis, tudo.
Como fazer:
1️⃣
Toque em "Gerar arquivo para compartilhar" abaixo
2️⃣
Um arquivo HockesJca_compartilhar.html vai baixar
3️⃣
Mande pelo WhatsApp, Google Drive ou e-mail
4️⃣
A Jack abre o arquivo no celular — dados já aparecem
Inclui todos os seus dados atuais dentro do arquivo
O backup salva seus dados num arquivo .json separado. Guarde no Google Drive ou mande para si mesmo pelo WhatsApp. Se o app perder os dados, você restaura por aqui.
Recomendado: faça backup toda semana
⚠️ Restaurar substitui todos os dados atuais pelo que está no arquivo selecionado. Só use se perdeu os dados ou quer transferir de outro celular.
Como restaurar:
1️⃣ Localize o arquivo .json ou .html de backup no seu celular
2️⃣ Toque em "Selecionar arquivo" abaixo
3️⃣ Confira o resumo e toque em "Restaurar"
📂
Selecionar arquivo de backup
Aceita .json ou .html com dados
🗄️ Onde ficam os dados hoje?
Os dados ficam numa "gaveta secreta" do navegador do seu celular, chamada localStorage. Funciona como um bloco de notas que só aquele navegador consegue ler.
⚠️ Quando posso perder os dados?
• Limpar cache/dados do Chrome ou Safari
• Trocar de celular sem fazer backup
• Abrir o .html numa aba anônima (privativa)
• Mandar o arquivo pro WhatsApp da Jack (dados não viajam junto, só o app vazio)
✅ Como proteger?
• Use 📲 Compartilhar para mandar com dados para a Jack
• Use 💾 Salvar backup toda semana — guarde no Google Drive
• O app avisa quando fizer mais de 7 dias sem backup
📊 O que é o CSV?
CSV é uma planilha simples que abre no Excel ou Google Sheets. Serve para importar dados em lote (ex: lançar 30 gastos de uma vez sem digitar um por um) ou para analisar os dados fora do app.
